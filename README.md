A full-stack web application designed to provide a supportive, accessible virtual therapy environment for users seeking mental health support and well-being resources.

## About
HealNest is a virtual therapy platform developed as a final year project to help users access mental health support in an intuitive, modern web interface. Inspired by real-world mental health platforms that help connect individuals with emotional support and therapeutic tools online, the platform aims to combine empathy-focused design with practical features for well-being.
This project includes both frontend and backend codebases to deliver interactive user experiences along with API and database integration.

## Features
- User Authentication: Secure role based signup for users and therapists.
- Therapy Dashboard: Personalized user dashboard to manage sessions and details.
- Booking Management: Schedule virtual therapy sessions.
- Google Calendar Integration: Automatic event creation and link creation upon booking for session.
- Mood Tracker: Tracker to track your mood daily basis.
- Payment Gateway: Secure payment.
- Forum Page: Anonymous forum page to share personal mental experience.

## Tech Stack
**Frontend:** HTML, CSS, React.js, MUI.
**Backend:** Node.js, Express.js
**Database:** MongoDB
